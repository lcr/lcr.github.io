## Headers

```
# This is an h1
## This is an h2  
### This is an h3
#### This is an h4
##### This is an h5
###### This is an h6
```

Horizontal line: `---`

break `2 spaces`

_italic_ `_italic_` or `*italic*`

__bold__ `__bold__` or `**bold**`

img `![alt](https://example.jpg)`

link [link](https://lcr.github.io/) `[link](https://example.com/)`  or  `[link](https://example.com/){:target="_blank"}`
